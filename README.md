# Codex over EasyTier

> I thought EasyTier was enough. Then Codex taught me about loopback addresses, SSH forwarding, and process environments.

This is a sanitized, story-driven guide to using VS Code Remote-SSH and Codex on a private Ubuntu server through a laptop's Clash proxy.

- [中文：从“连不上”到稳定运行](README.zh-CN.md)
- [English: From “Connection Failed” to a Stable Remote Setup](README.en-US.md)

The public version deliberately omits real IP addresses, host fingerprints, usernames, network names, passwords, EasyTier secrets, tokens, and private-key contents. Those details are operational credentials or useful targeting information, not documentation.
